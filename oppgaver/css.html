Stiling med CSS

Foreløpig ser nettsiden vår ganske kjedelig ut, bare svart text på hvit bakgrunn. Det er fordi HTML bare definerer hva siden skal inneholde, og ikke hvordan det skal se ut. For å bestemme hvordan siden skal se ut bruker vi et annet språk som kalles CSS. (Det er en forkortelse for Cascading Style Sheets.) CSS brukes altså til å stile en webside, altså styre hvordan den skal se ut.

Inline CSS

Den enkleste måten å bruke CSS på er å bruke style-attributten direkte på html-elementene. Dette kalles inline CSS Det lett å lære og enkelt når vi ikke har så mye CSS men kan bli veldig uoversiktlig når det begynner å bli mye. Senere skal vi lære å plassere CSS i en egen fil, men vi begynner med den enkle måten først.

CSS består av attributter og verdier. F.eks kan vi bruke attributten "color" til å sette fargen på noe og 'text-align' til å midtstille tekst slik som dette: 

<h1 style="color: red; text-align: center;">Velkommen til Kodeklubben!</h1>

Dette vil vises som

Velkommen til Kodeklubben!

Vi setter altså CSS på et element ved å bruke 'style' attributten og så liste opp CSS attributter inni den. Vi kan ha så mangge CSS attributter vi vil på hvert element, men hvis det blir veldig mange er det lurere å flytte dem til en egen fil. Det skal vi lære litt senere. Hver CSS attributt er bygget opp slik: 

 attributt: verdi;

CSS arves av alle elementene som er inni det elementet som CSS'en er satt på. Samtidig overstyres CSS attributter hvis de også er satt på et element som er inni.

        <div style="text-align: center; color: red">
          <p>
            Første avsnitt
          </p>
          <p style="color: blue">
            <span>Andre avsnitt</span>
          </p>
        </div>

Vises som:

Første avsnitt

Andre avsnitt

Begge avsnittene inni div-elementet arver stilingen fra div'en. Derfor blir begge midtstilt. Det første avsnittet arver også den røde teksten, men på det andre avsnittet har vi sagt at teksten skal være blå i stedet. Så selv om vi arvet rød tekst overstyrer vi den til å bli blå i stedet. Hvis vi hadde hatt flere elementer inni det andre avsnittet, ville de arvet den blå fargen i stedet for den røde.

Fordi CSS attributtene overstyrer hverandre, kan ofte være vanskelig å finne ut hvorfor det blir som de blir når vi har mye CSS. Da er det veldig nyttig å bruke utviklerverktøyet som finnes i de fleste nettlesere (f.eks Chrome Developer Tools) til å undersøke hvilken CSS som er gjeldende for et element og hvor denne kommer fra.

Utviklerverktøyet er også veldig nyttig til å prøve forskjellige attributter og verdier og se hvordan ting vil se ut. Prøv det!

De vanligste CSS attributtene

Det finnes veldig mange CSS attributter og det finnes folk som jobber bare med å skrive CSS. Vi kan ikke gå gjennom alle her, men vi skal lære noen av de vanligste. For å lære flere CSS attributter og mer om CSS kan man finne masse informasjon på internett. F.eks på w3schools eller Mozilla Developer Network

Her er noen av den vanligste CSS attributtene:

Attributt Bruksområde Eksempel
color Setter forgrunnsfargen. Noen farger kan angis med navn. For andre farger må vi bruke fargekoder color: red;
color: #ff0000;
background-color Setter bakgrunnsfargen. Hvis vi setter denne på <body> settes fargen på hele siden. background-color: red;
bacjground-color: #ff0000;
background-image Brukes til å sette et bilde som bakgrunn på <body> elementet. Bildet gjentas mange ganger til det dekker hele bakgrunnen. Det finnes flere attributter som styrer bakgrunnsbildet f.eks background-position, background-size og background-repeat. Slå opp disse på w3schools eller MDN for å finne ut mer om dem. background-image: url("bakgrunn.jpg");
font-size Setter størrelsen på teksten. Settes som regel i piksler (px) eller punkter (pt) font-size: 12px;
font-size: 22pt;
text-align Justerer teksten til høyre, venstre eller midtstilt. text-align: left;
text-align: right;
text-align: center;
text-align: justify;
border Brukes til å tegne en ramme rundt et element. Brukes ofte på et <div> element. De finnes mange andre attributter som begynner med border- som lar oss styre i detalj hvordan rammen skal se ut. Disse kan du finne selv på w3schools eller MDN. Border-attributten tar tre parametre: border-width, border-style og border-color. Hver av disse er egne attributter som kan brukes hver for seg i stedet for border. border: 1px solid blue;
border: 2px dotted red;
border: medium dashed green;
padding Padding betyr polstring og brukes til å lage litt "luft" rundt innholdet i et element. Hvis elementet har en border, vil paddingen ligge på innsiden av den. Vi kan ha padding på alle fire sidene eller bare på noen av dem. Padding-attributten tar 1 til 4 parametre. Hvis vi bruker bare 1 blir det like mye padding på alle fire sidene. Hvis vi bruker to er den første toppen og bunnen, og den andre er høyre og venstre. padding: 10px;
padding: 10px 20px;
padding: 10px 20px 30px 40px;
margin Margin ligner på padding, men brukes til å lae "luft" mellom dette elementet og andre elementer. Hvis elementet har border vil marginen ligge på utsiden av den. Vi kan bruke 1 til 4 parametre som fungerer på samme måten som for padding. margin: 10px;
margin: 10px 20px;
margin: 10px 20px 30px 40px;
height og width Setter høyden og bredden på elementet. De kan brukes hver for seg. Hvis bare den ene er satt vil den andre kanskje beregnes automatisk. Hvis vi f.eks setter bare height på et bilde, vil width justeres automatisk slik at hele bildet blir større. height: 20px
width: 100%
height: auto
float Brukes til å få et element til å "flyte". At det flyter vil si at det legger seg til høyre eller venstre, og at tekst og andre elementer "flyter" rundt elementet. Det brukes til å f.eks. plassere et bilde i en artikkel og så få teksten til å plassere seg fint rundt bildet. Float brukes også til layout, f.eks. plassere en navigasjonsmeny helt til venstre og så selve sideinnholdet ved siden av. I så fall bruker vi float: left; på menyen. 
  <div style="float: left;">
    Menyen her
  </div>
  <div>
    Sideinnhold
  </div>

float: left;
float: right;
float: none;

Prøv deg frem med CSS

Nå når vi har lært noen av de vanligste CSS attributtene kan du prøve å bruke dem på siden din.

  • Prøv å sette farger på forskjellige ting.
  • Prøv å sette en bakgrunnsfarge på selve siden.
  • Prøv å sette et bakgrunnsbilde, og se hva som skjer.
  • Prøv å ekperimentere med de andre background-attributtene som background-position, background-size og background-repeat. Se om du får et bilde til å dekke hele siden.
  • Prøv å bruke border på f.eks. en <div>. Hva skjer? Prøv å endre størrelsen på div-en med height og width.
  • Prøv deg frem med margin og padding. Husk at markin lager mellomrom på utsiden av elementet, mens padding lager mellomrom inni elementet. Prøv å plassere teksten din på en fin måte ved å bruke padding og margins. Legg gjerne til ekstra <div> elementer for å få det sånn som du vil.
  • Prøv å lage en liten meny til venstre på siden ved å bruke float. Lag en <div> som inneholder menyen og sett "float: left;" på den. Plasser selve innholdet i en ny <div> like etter den første, og se hva som skjer.

Det neste vi skal lære er tabeller og lister

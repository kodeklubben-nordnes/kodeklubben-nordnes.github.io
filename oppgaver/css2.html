Mer CSS

Hittil har vi skrevet CSS i style-attributten på HTML elementene. Dette går fint når vi ikke har så mye CSS og ikke skal gjøre så store endringer. Det er en enkel måte å komme igang med CSS, men det er ikke måten det vanligvis gjøres på. CSS kan også skrives for seg selv og ikke blandet sammen med HTML-koden. Da kan CSS-koden plasseres enten i en egen fil, eller i et <style> element i <head>. Måten vi skriver CSS på er den samme i begge tilfellene. Vi kan begynne med å plassere koden i <head> siden det er enklere å holde oversikten når alt er på samme sted. Når det begynner å bli mye kan vi enkelt flytte CSS-koden til en egen fil. CSS-filer gjør det også enkelt å bruke stiling som andre har laget

CSS-blokker

Vanlig CSS (altså CSS som ikke er i en style-attributt) ser slik ut:

  selektor {
    css-attributt: verdi;
    css-attributt: verdi;
  }

Dette er et eksempel for å vise hvordan strukturen ser ut. Vi ser at vi begynner med noe vi kaller en selektor. Så har vi et par krøllparenteser og i mellom dem har vi helt vanlig CSS slik som vi er vant til. Så i stedet for å skrive CSS i en style-attributt på et HTML-element skriver vi det i mellom to krøllparenteser med en selektor foran. Dette kaller vi en CSS-blokk.

Selektoren

Selektoren avgør hvilke elementer CSS-blokken skal gjelde for. Det finnes flere typer selektorer. Den enkleste er å bruke navnet på et HTML-element. Hvis vi f.eks. vil at alle h1-elementer skal være røde kan vi gjøre dette:

  h1 {
    color: red;
  }

Dette vil gjøre at alle h1-elementer på hele siden vil bli røde. Dette får samme effekt som å gjøre dette: 

  <h1 style="color: red;">Overskrift 1</h1>
  <h1 style="color: red;">Overskrift 2</h1>

Vi skriver altså ikke CSS på hvert enkelt element slik vi har gjort før. I stedet deler vi CSS-koden inn i blokker og bruker selektoren til å bestemme hvilke elementer blokken gjelder for. Det gjør at vi ikke trenger å gjenta CSS-koden så ofte. Det holder å si en gang at alle overskrifter skal være røde, og så trenger vi ikke å tenke mer på det.

Det finnes flere typer selektorer. De vanligste er

  • Navnet på HTML-elementer
  • ID-en til et HTML-element.
  • CSS klasser

Vi skal gå gjennom hver av dem og forklare hvordan de virker.

Element-selektor

Element-selektoren er den vi brukte i eksempelet over. Den er enkel og grei. Det er bare navnet på et html-element. Alle html-elementer kan brukes, og selektoren vil gjelde for alle elementer av den typen.

Eksempel:

  h1 {
    color: red;
  }

ID-selektor

ID er en egen attributt som kan settes på alle HTML-elementer. ID-er må være unike på en side. ID settes slik:

  <h1 id="overskrift1">Overskrift 1</h1>
  <h1 id="overskrift2">Overskrift 2</h1>

Da kan ID-selektoren brukes ved at vi skriver navnet på ID-en med # foran, slik som dette:

  #overskrift1 {
    color: red;
  }

  #overskrift2 {
    color: blue;
  }

Da vil bare overskrift1 bli rød og bare overskrift2 bli blå. CSS-en gjalder altså bare for akkurat de elementene som har den ID-en.

Klasse-selektor

En CSS-klasse er også en egen HTML-attributt som kan settes på alle HTML-elementer. Men i motsetning til ID trenger ikke klasser å være unike og vi kan også ha flere klasser på hvert HTML-element. Klasser settes slik:

  <h1 class="red">Overskrift</h1>
  <p class="blue">Litt blå tekst</p>
  <p class="red">Litt rød tekst</p>

Da kan klasse-selektoren brukes ved at vi skriver navnet på klassen med . (punktum) foran, slik som dette:

  .red {
    color: red;
  }

  .blue {
    color: blue;
  }

Navnet på klassene finner vi på selv. Det er lurt å lage navn som passer med det CSS-en gjør, og så ha en blokk for hver klasse. I dette tilfellet har vi en klasse som gjør ting røde, og en som gjør ting blå. Disse klassene kan nå brukes på mange forskjellige HTML-elementer for å gjøre dem røde eller blå.

Avanserte CSS selektorer

CSS-selektorer kan være veldig avanserte. Vi kan f.eks bruke flere selektorer samtidig for å velge bare elemeter som er inni et annet element, eller elementer som har flere klasser, eller elementer som er av en gitt type OG en gitt klasse. Dette skal vi løre mer om senere, men hvis du vil lære mer om dette nå med en gang finnes det en liste med alle variantene her.

CSS i <head>

Vi kan skrive CSS direkte i HTML filen ved å plassere den i et <style> element inni <head>, slik som dette:

  <head>
    <style>

      .red {
        color: red;
      }

      .blue {
        color: blue;
      }

    </style>

  </head>
